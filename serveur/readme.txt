formulaire avec mise à jour des valeurs via REST API (nom, prenom et all) utilisation de GET et POST

pour lancer le serveur : 
node serveur.js

Puis aller sur un navigateur et dans l'url taper : http://127.0.0.1:3000
cela va lancer le code index.html (dans public) qui va mettre à jour les valeurs de nom et prenom via POST
on pourra ensuite venir lire les valeurs de nom et prenom : http://127.0.0.1:3000/all
